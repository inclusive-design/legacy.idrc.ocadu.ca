This tutorial will discuss the basics of the reading toolbar in the Kurzweil 3000 program. This toolbar will give users control over the way the Kurzweil program reads. The tutorial will introduce users to the buttons on the toolbar and explain a little about each.

  1. Open the Kurzweil 3000 program and log in under a user name. Make sure that the reading toolbar is on the screen and is active.

The buttons on the toolbar

Reading Unit - The first drop down box (it is on the right side of the word \'by\') controls the reading unit. This controls how much text is highlighted at one time while the program is reading.

Words per minute - The command to the left of the word \'at\' controls the number of words per minute that are read. The higher the number of WPM, the faster the text is read.

Zoom - This feature controls the percentage at which the text on the screen is magnified. The higher the number , the bigger the text will appear on the screen.

Page - This command tells the user what page of the document he/she is working on. The small grey button on the right of the word \'page\' tells the user what page they are currently working on. The buttons with the two red arrows (one pointing up and the other pointing down) allow the user to navigate through the pages of the document by single clicking on either one. Clicking on the textbox with the number, the user can type in the number of the page that he/she would like to go to. The small grey box at the end of this command tells the user the total number of pages that are in the document.

Spell Check - The button with the \'ABC\' on top of the check mark is the spell checker. This function scans the document and looks for spelling mistakes. If it finds any, it will ask the user if he/she wants to change them.

Word Prediction - The button with the ball on top of the square is the word prediction function. When on, it attempts to predict a word as a user is tying it. To select a word, hold down the control button and press the number that corresponds to the word.

Alignment - The first drop down toolbar controls the alignment of the text. To align the text, highlight it, click the alignment drop down menu and click on the alignment that you would like. (left, right or center)

Writing Font Size - The last drop down menu controls the size of the font. To change the font\'s size, highlight the font you would like to work with , click the font size drop down menu and choose the size you would like the font to be.
